Estructura

Luego de declarar el tipo de documento, debemos construir la estructura HTML. < html>. Este elemento cercará al resto del código:

  • < !DOCTYPE html>
  • < html lang=”es”>
  • < /html>
    • El elemento < head > va primero e igual que el resto de los elementos estructurales tiene una etiqueta de apertura y una de cierre:
  • < !DOCTYPE html>
  • < html lang="es">
  • < head>
  • < /head>
  • < /html>
    • El cuerpo representa la parte perceptible de todo documento y desarrollado previas de HTML:
  • < !DOCTYPE html>
  • < html lang="es">
  • < head>
  • < /head>
  • < body >
  • < /body>
  • < /html>
    • construir la cabecera del documento. Ésta es una etiqueta < meta> que especifica cómo el texto será presentado en pantalla:
  • < !DOCTYPE html>
  • < html lang="es">
  • < head>
  • < meta charset=”iso-8859-1”>
  • < / head>
  • < body>
  • < / body>
  • < / html>
    • El invento de este elemento en HTML5, es solo reducción. La nueva etiqueta < meta> para la definición del tipo de caracteres es más corta y simple. podemos cambiar el tipo iso-8859-1 por el necesario para nuestros documentos y agregar etiquetas como description o keywords para definir otros aspectos de la página web:
  • < !DOCTYPE html>
  • < html lang="es">
  • < head>
  • < meta charset="iso-8859-1">
  • < meta name=”description” content=”Ejemplo de HTML5”>
  • < meta name=”keywords” content=”HTML5, CSS3, Javascript”>
  • < / head>
  • < body>
  • < / body>
  • < / html>
    • La etiqueta < title>, simplemente detalla el título del documento:
  • < ! DOCTYPE html>
  • < html lang="es">
  • < head>< /head>
  • < meta charset="iso-8859-1">
  • < meta name="description" content="Ejemplo de HTML5">
  • < meta name="keywords" content="HTML5, CSS3, JavaScript">
  • < title>Este texto es el título del documento
  • < / head>
  • < body>
  • < / body>
  • < / html>
    • Este elemento es usado para incorporar estilos, códigos Javascript, imágenes o iconos desde archivos externos. También incorpora los archivos con estilos CSS:
  • < ! DOCTYPE html>
  • < html lang="es">< /html>
  • < head>< /head>
  • < meta charset="iso-8859-1">
  • < meta name="description" content="Ejemplo de HTML5">
  • < meta name="keywords" content="HTML5, CSS3, JavaScript">
  • < title>Este texto es el título del documento
  • < link rel=”stylesheet” href=”misestilos.css”>
  • < / head>
  • < body>
  • < / body>
  • < / html>
