EL ELEMENTO Y SUS ETIQUETAS

Las etiquetas tienen gran valor en un documento HTML por que ofrecen información especial que es utilizada para determinar 
la representación final del documento. Esto significa que donde se define una etiqueta en el código, algo ocurrirá en la 
representación del documento, unas veces visualmente y otras de modo oculto. Todas las etiquetas comparten el mismo formato: 
principian con el signo menor que "<" y terminan con el signo mayor que ">".
el elemento html tiene dos etiquetas:
Etiqueta de inicio < html> define el inicio del documento HTML.
Etiqueta de cierre < / html> define el final del documento HTML.


LOS ATRIBUTOS Y SUS VALORES


Son las propiedades que se le pueden fijar a los elementos.
A html se le puede asignar un atributo para mostrar en qué idioma está escrito el documento (lang) y podemos asignarle 
como valor un código de idioma ("es" para el español, "en" etc)
< html lang='es'>...</ html>
Un elemento puede tener distintos atributos:
< elemento atributo='valor' atributo='valor'> ... < / elemento>
Los atributos deben estar en la etiqueta inicial, el orden de los atributos dentro de la etiqueta es indiferente.
Deben apartarse con un espacio en blanco del nombre del elemento y de otros atributos. Atributo y valor deben 
estar unidos por el signo:  =  Los valores deben estar encerrados con comillas, dobles o simples.
