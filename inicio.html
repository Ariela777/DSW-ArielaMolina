Objetivo General

Desarrollar sitio web en formato HTML5 y CSS3 para ser visualizados en diferentes navegadores teniendo en cuenta la funcionalidad y la usabilidad, por medio del uso de diferentes editores de codigo propuestos en el curso de Programaciòn de Diseño Web


Objetivos Especificos

  • Objetivo Especifico 1: Desarrollar sitio web en formato HTML5 y CSS3
  • Objetivo Especifico 2: Desarrollar sitio web como un OVI Objeto Virtual de Informaciòn
  • Objetivo Especifico 3: Aplicar estructura bàsica de de CCS3
  • Objetivo Especifico 4: Aplicar elementos de CSS3 al sitio Web


Contenido

Bienvenidos a nuestro curso de HTML5, un curso en el que conoceremos el lenguaje de marcas o etiquetas más afamado del mundo para aprender a hacer nuestra primera página web. En este curso enseñaremos lo más básico de HTML para poder iniciar la creación de páginas web. Para arrancar con este curso introductorio no es necesario tener ningún conocimiento de HTML, tampoco es obligatorio conocer otro lenguaje de programación. Solo necesitamos motivación por el aprendizaje.
